# Lancer le projet The Descent

## Voici les informations nécessaires pour lancer ce projet :

* Votre système d'exploitation doit préférablement être Windows 10 ou 11. Le projet fonctionne également sur MacOS et Linux, mais ce ne sont pas les OS recommandés.
* Il est recommandé d'avoir au moins 8 Go de RAM disponible sur votre PC.
* Vous aurez besoin de la version 3.13 de python. Le projet est prévu pour être lancé avec cette version, il a été développé et testé avec.
* Pour faire fonctionner le projet, vous devez installer les librairies pygame (version 2.6.1) et matplotlib (version 3.10.0). Vous pouvez les installer avec la commande *pip install -r requirements.txt*.
* Lancez ensuite le fichier `sources/main.py` avec cette configuration.
