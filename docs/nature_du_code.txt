- Modalités d'utilisation de l'intelligence artificielle :
    - L'usage de l'IA a été limité, réfléchi et exclusivement pédagogique :
        - Seuls deux fichiers contiennent des portions de code directement générées par ChatGPT ("sources/file_type_manager.py" et "test/test_multiprocessing/chargement.py")
        - Dans tous les autres fichiers du projet, aucun code généré par IA n'a été copié-collé. Nous avons uniquement demandé des conseils de fonctionnement ou d'utilisation de
          bibliothèques, l'implémentation finale est toujours de notre main.

    - Nous n'avons eu recours à aucune intelligence artificielle (ni conseil ni code) pour les fichiers suivants, dont les idées et implémentations sont entièrement originales :
        - "sources/utils.py"
        - "sources/simulation.py"
        - "sources/environnement/ia/main.py"

    - Justification de l'usage de l'IA :
        - L'IA a été utilisée uniquement pour gagner en rapidité et en clarté sur des points techniques que nous ne maîtrisions pas encore (fonctionnement de bibliothèques, interactions
          avec l'OS ou gestion de fichiers externes). Cela nous a permis de nous concentrer sur la logique du projet plutôt que sur la syntaxe de modules peu documentés. Nous n'avons
          jamais utilisé l'IA pour concevoir l'architecture, les algorithmes ou les fonctionnalités centrales du simulateur.
